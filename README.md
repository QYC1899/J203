# J203 班级记录

班级信息管理应用，包含学生列表、教师信息等功能。

## 技术栈

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## 本地开发

```sh
npm i
npm run dev
```

## 在 GitHub Codespaces 中开发

1. 前往 GitHub 仓库主页，点击绿色的 **Code** 按钮
2. 选择 **Codespaces** 标签，点击 **Create codespace on main**
3. 等待 Codespace 环境加载完成（无需额外安装任何工具，Node.js 和 npm 已预装）
4. 在左侧文件资源管理器中，右键点选项目根目录，选择 **"Open in Integrated Terminal"** 打开集成终端
5. 在终端中运行以下指令：

```sh
npm i
npm run dev
```

6. Codespace 会自动检测到端口并弹出提示，点击 **"Open in Browser"** 即可预览应用
